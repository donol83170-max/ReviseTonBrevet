Questions fréquentes — Orthographe

Comment distinguer 'a' et 'à' ?
Remplace par 'avait'. Si ça marche, c'est 'a' (verbe avoir). Si ça ne marche pas, c'est 'à' (préposition). Ex : il a faim → il avait faim ✓ → 'a'.
C'est quoi l'accord du participe passé avec avoir ?
Avec avoir, le participe s'accorde avec le COD seulement s'il est placé AVANT le verbe. Ex : 'les sons qu'il a composés' (COD avant) vs 'il a composé des sons' (COD après = pas d'accord).
Comment mettre au pluriel les noms composés ?
Ça dépend de la structure : nom+nom = les deux prennent le pluriel (chefs-lieux), verbe+nom = seul le nom varie (tire-bouchons), préposition+nom = seul le nom varie (avant-gardes).

🔄 Les homophones — le grand classique

Des mots qui se prononcent pareil mais s'écrivent différemment. C'est le piège préféré des correcteurs. Même le 67 Elephant s'y perd !

a
verbe avoir (il a)
"Jul a sorti un album."
à
préposition
"Il pense à ses fans."
est
verbe être (il est)
"SCH est talentueux."
et
conjonction de coordination
"Jul et SCH rappent."
son / sa / ses
adjectifs possessifs
"Son flow est incroyable."
sont
verbe être au pluriel
"Ils sont géniaux."
ou
conjonction (choix)
"SCH ou Jul ?"
où
adverbe de lieu / relatif
"Le studio où il chante."
ce / cet / cette
adjectifs démonstratifs
"Ce son est fort."
se
pronom réfléchi
"Il se prépare."
mais
conjonction d'opposition
"Rapide mais précis."
mes
adjectif possessif pluriel
"Mes sons préférés."

✅ Participe passé — les règles

Avec être : le participe s'accorde avec le SUJET.
Ex : "Les rappeurs sont arrivés." (masc. pluriel)
Avec avoir : le participe s'accorde avec le COD placé AVANT le verbe.
Ex : "Les sons qu'il a composés." (COD "que" = sons, masc. plur.)
Ex : "Il a composé des sons." (COD après = pas d'accord)

📚 Pluriel des noms composés

Les noms composés ont des règles de pluriel spécifiques. Voici les cas à connaître :

  • Nom + nom : les deux prennent le pluriel → des chefs-lieux
  • Nom + adjectif : les deux prennent le pluriel → des beaux-arts
  • Verbe + nom : seul le nom prend le pluriel (si sens le permet) → des tire-bouchons, des abat-jour
  • Préposition + nom : seul le nom varie → des avant-gardes
💡 Astuce homophones : Pour "a/à" : remplace par "avait". Si ça marche → "a" (verbe). Si ça ne marche pas → "à" (préposition). Même logique : "est/et" → remplace par "était". "son/sont" → remplace par "leur/leurs".
